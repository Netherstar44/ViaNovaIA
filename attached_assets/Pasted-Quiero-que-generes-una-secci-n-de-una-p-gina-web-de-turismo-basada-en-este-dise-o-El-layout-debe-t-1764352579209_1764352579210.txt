Quiero que generes una sección de una página web de turismo basada en este diseño:

El layout debe tener dos columnas principales:

Columna izquierda: un mapa grande ocupando la parte superior

Debajo del mapa: un recuadro con información dinámica del lugar seleccionado (nombre, foto, descripción, horario, etc.)

Columna derecha: un panel vertical dividido en secciones:

“Hoteles”

“Restaurantes”

“Sitios recreativos o turísticos”

“Transporte”

Cada sección debe mostrar varias tarjetas pequeñas (cards) con imagen y nombre del lugar.

🔷 Características específicas que debe generar
🔹 1. Mapa

Debe colocarse en la parte izquierda superior.

Puede ser Google Maps, Mapbox o Leaflet.

Debe permitir seleccionar un marcador.

Al seleccionar un marcador, debe actualizar automáticamente el cuadro de información debajo del mapa.

🔹 2. Información debajo del mapa

Debe incluir:

Imagen principal del lugar

Nombre

Categoría (Hotel, Restaurante, Sitio recreativo)

Descripción corta

Botón “Ver más”

Este cuadro debe ser responsivo y adaptarse al tamaño del contenedor.

🔹 3. Panel derecho con categorías

El panel derecho debe estar dividido en secciones visuales:

Título (por ejemplo: “Hoteles”)

Lista de tarjetas en filas o columnas con:

Imagen pequeña

Nombre del lugar

Breve etiqueta opcional

Las tarjetas deben ser clickeables y al hacer clic deben mostrar su información en el área debajo del mapa.

🔹 4. Estilo y distribución

Diseño limpio y organizado.

El mapa debe ocupar el 60% del ancho.

El panel derecho debe ocupar el 40% restante.

Se debe poder implementar en TypeScript, HTML, CSS o React.

Que sea responsivo para pantallas pequeñas.

🔷 5. Lo que quiero que generes

De acuerdo con lo anterior, genera:

El HTML completo de esta sección

El CSS responsivo

El JavaScript/TypeScript para:

Cargar el mapa

Manejar clics en marcadores

Actualizar la información del lugar

Manejar clics en las tarjetas del panel derecho

Si usas React o TypeScript, genera los componentes modulares.
Si usas HTML/JS puro, colócalo organizado.

El resultado debe reproducir fielmente el diseño dibujado:
🔹 Mapa arriba a la izquierda
🔹 Información abajo del mapa
🔹 Categorías verticales a la derecha
